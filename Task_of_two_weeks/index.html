HTML & CSS Basics Summary

HTML Summary

    HTML(Hypertext Markup Language) is used to define your page contect,structure and meaning.
    You don't use it for styling purposes.Use CSS for that instead

  1. HTML uses elements to describe (annotate) content.
  2. HTML elements typically have an opening tag,content and a closing tag.
  3. You can also configure elements with attributes
  4. There is a long list of elements but you will gain experience over time, no need to worry

    5. Learn more about html elements on the MDN HTML element reference

CSS Summary

    (CSS) Cascading Style Sheets is used for styling your page.

  1. Styles are assigned via property-value pairs
  2. You can assign styles via the 'style' attribute
  3. To avoid code duplication, you typically use global styles(via the 'style' element) instead
  4. Alternatively, you work with external stylesheet files which you 'link' to
  5. When working with CSS, concepts like "inheritance" and "specificity" should be understood

    6. Learn more about all available CSS properties and values on the the MDN CSS property reference
